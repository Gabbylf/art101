Challenge

The challenge was to format an function in javascript. With that, we also learned how to split apart, reverse and put back together something the user on our website inputs in by themselves with no control over the website or the coders. Lastly, we were tasked to finally make our own html page.

Problems

The biggest problem we faced was sorting out our javascript so it actually read and posted on the website. However, through numerous trial and error, we did eventually get it to work and respond to the user input as well!

Reflection
